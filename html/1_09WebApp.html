移动设备设置 <meta name="viewport" content="width=device-width,initial-scale=1.0,user-scalable=no" />
视口 name="viewport"

初始的缩放比例为1:1 initial-scale=1.0
禁止对页面进行缩放 user-scalable=no
淘宝弹性布局 <meta name="viewport" content="width=device-width, initial-scale=0.5, user-scalable=no" />

单位换算
描 述 代 码
px 逻辑像素，浏览器使用的抽象单位
dp,pt 设备无关像素
dpr 设备像素缩放比
计算公式 1px = dpr * dpr * dp
dpi 打印机每英寸可以喷的墨汁点（印刷行业）
ppi 屏幕每英寸的像素数量，即单位英寸内的像素密度
设备分辨率计算方法 iphone4为例：√(960^2+640^2) / 3.5英寸 = 330 ppi
iphone设备宽度
设 备 主屏分辨率 屏幕像素密度 设备像素缩放比 设备宽度
iphone4 640*960 330ppi dpr=2 320*480
iphone5 640*1136 326ppi dpr=2 320*568
iphone6 750*1334 326ppi dpr=2 375*667
iphone6plus 1080*1920 401ppi dpr=3 414*736
@media
描 述 代 码
小屏幕 @media (min-width: @screen-sm-min) { ... }平板，大于等于 768px
中等屏幕 @media (min-width: @screen-md-min) { ... }桌面显示器，大于等于 992px
大屏幕 @media (min-width: @screen-lg-min) { ... }大桌面显示器，大于等于 1200px
大屏幕 @media screen and (min-width:1200px){ ... } 大桌面显示器，大于等于 1200px
flex
描 述 代 码
display display: flex;弹性布局
flex-direction row/row-reverse/column/column-reverse 方向
flex-direction: row; /*default*/行，正常
flex-direction: row-reverse;行，倒序
flex-direction: column;列，正常
flex-direction: column-reverse;列，倒序
flex-wrap nowrap/wrap/wrap-reverse 换行
flex-wrap: nowrap; /* default */不可换行
flex-wrap: wrap;可换行
flex-wrap: wrap-reverse;可换行，倒序
justify-content flex-start/flex-end/center/space-between/space-around 水平对齐
justify-content: flex-start; /* default */左对齐
justify-content: flex-end;右对齐
justify-content: center;水平居中
justify-content: space-between;两边对齐
justify-content: space-around;两边居中
align-items flex-start/flex-end/center/baseline/stretch 垂直对齐
align-items: flex-start;顶对齐
align-items: flex-end;底对齐
align-items: center;垂直居中
align-items: baseline;基线
align-items: stretch; /* default */拉伸
align-self flex-start/flex-end/center/baseline/stretch/auto 垂直对齐
align-self: flex-start;顶对齐
align-self: flex-end;底对齐
align-self: center;垂直居中
align-self: baseline;基线
align-self: stretch;拉伸
align-self: auto; /* default */拉伸
