css3兼容

transform（转换）、transition（过渡）、@keyframes（关键帧）、animation（动画）
添加前缀 兼容浏览器
-webkit- Safari and Chrome（苹果、谷歌）
-moz- Firefox（火狐）
-ms- IE9（IE浏览器）
-o- Opera（世界之窗）

transform（转换）

scale（伸缩）、skew（倾斜）、translate（移动）、rotate（旋转）、rotateX（3D旋转）、rotateY（3D旋转）、matrix（整体转换）
属 性 声 明描 述
scale transform: scale(3,6);宽度变为 3 倍，高度变为 6 倍（伸缩）
skew transform: skew(30deg,60deg);围绕 X 轴翻转 30 度，围绕 Y 轴翻转 60 度（倾斜）
translate transform: translate(30px,60px);左移 30 px，上移 60 px（移动）
rotate transform: rotate(30deg);顺时针旋转 30 度（旋转）
rotateX transform: rotateX(60deg);3D水平顺时针旋转 60 度（3D水平旋转）
rotateY transform: rotateY(90deg);3D垂直顺时针旋转 90 度（3D垂直旋转）
matrix transform:matrix(1,0,0,1,0,0);伸缩X、倾斜X、倾斜Y、伸缩Y、移动X、移动Y（整体转换）

transition（过渡）

property（css属性）、duration（用时长度）、timing-function（过渡效果）、delay（开始时间）
属 性 声 明描 述
transition transition: transform 2s linear 1s;transform，用时2s，匀速，1s后开始
transition-property transition-property:transform属性名称：transform（属性名称）
transition-duration transition-duration: 2s;用时长度（默认为0）
transition-timing-function transition-timing-function: linear;过渡效果（linear匀速、ease曲线，默认为ease）
transition-delay transition-delay: 1s;何时开始（默认为0）

@keyframes（关键帧）

@keyframes cssName{ } -webkit-等前缀在keyframes前面添加
属 性 声 明描 述
from-to @keyframes cssName{
    from {background: red;}
    to {background: yellow;}
} 动画名称：cssName
0%-100% @keyframes cssName{
    0% {background: red;}
    25% {background: yellow;}
    50% {background: blue;}
    100% {background: green;}
}

animation（动画）

name（动画名称）、duration（用时长度）、timing-function（过渡效果）、delay（开始时间）、 iteration-count（播放次数）、direction（顺逆播放）、play-state（运行或暂停）
属 性 声 明描 述
animation animation: cssName 2s linear 1s infinite alternate;transform，用时2s，匀速，1s后开始
animation-name animation-name: className;动画名称：cssName（定义动画名称）
animation-duration animation-duration: 2s;用时长度（默认是 0）
animation-timing-function animation-timing-function: linear;过渡效果（linear匀速、ease曲线，默认为ease）
animation-delay animation-delay: 1s;何时开始（默认为0）
animation-iteration-count animation-iteration-count: infinite;播放次数（infinite永远，默认为1）
animation-direction animation-direction: alternate;顺逆播放（alternate反向，默认为normal）
animation-play-state animation-play-state: running;运行或暂停（paused暂停、running开始，默认为running）

css3属性

css3边框
属 性 声 明描 述
border-radius border-radius:25px;（边框圆角）
box-shadow box-shadow: 10px 10px 5px #888888;X轴、Y轴、模糊大小、颜色（边框阴影）
border-image border-image:url(bg.jpg) 30 30 round;背景、X轴、Y轴、重复性（边框背景）
css3背景
属 性 声 明描 述
background-size background-size:40% 100%;（像素或百分比伸缩）
background-origin background-origin:content-box;content-box、padding-box、border-box（定位区域）
background-clip background-clip:content-box;content-box、padding-box、border-box（绘制区域）
css3文本
属 性 声 明描 述
text-shadow text-shadow: 3px 2px 1px #f00;X轴、Y轴、模糊距离、颜色（文字阴影）
word-wrap word-wrap:break-word;对长单词进行拆分，并换行到下一行（英文换行）
CSS3字体
属 性 声 明描 述
@font-face @font-face{
    font-family: myFirstFont;
    src: url('Sansation_Light.ttf'),
    url('Sansation_Light.eot'); /* IE9+ */
}

div{
    font-family:myFirstFont;
} 定义字体的名称：myFirstFont
css3多列Internet Explorer 不支持 column-count 属性
属 性 声 明描 述
column-count column-count:3;（分隔的列数）
column-gap column-gap:40px;（列之间的间隔）
column-rule column-rule:3px outset #f00;（列之间的宽度、样式和颜色）
css3用户界面Internet Explorer 不支持 outline-offset 属性
属 性 声 明描 述
resize resize:both;
overflow:auto; （调整元素尺寸，horizontal宽、vertical高、both、none，配合overflow使用）
box-sizing box-sizing:border-box;（以特定的方式定义匹配某个区域的特定元素）
outline-offset div{
    border:1px solid black;
    outline:2px solid red;
    outline-offset:15px;
} 在div外15px处绘制一个2px的红色实线轮廓（绘制轮廓）
